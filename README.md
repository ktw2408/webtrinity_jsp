# webtrinity_jsp
기존 데모페이지의 HTML 코드를 JSP 코드로 변환하여 웹 컨테이너로 톰캣을 사용하여 
MVC1 기반으로 동적인 웹사이트를 구현한 팀 프로젝트 입니다.
데이터베이스는 mysql을 사용하였습니다.
